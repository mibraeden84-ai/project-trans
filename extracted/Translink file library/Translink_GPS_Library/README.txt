📡 TRANSLINK GPS LIBRARY
=========================
GPS configuration files and firmware for Translink fleet tracking.

BRANDS (one page per brand):
  Teltonika/   → FMB920, FMC920, FMB130, FMB125, FMB140, FMC150 + firmware
  GalileoSky/  → Base config, server config, V7 model + firmware
  StarLink/    → Base config, APN, advanced config + firmware
  Common_Settings/ → APN, server IP/port, SIM setup, installation guide
  Archive/     → Deprecated/old files

HOW TO USE:
  1. Open brand folder (Teltonika, GalileoSky, or StarLink)
  2. Read INDEX.txt for available files
  3. Download config for your device model
  4. Apply config via USB/Bluetooth to the GPS device
  5. Latest firmware updates in Firmware/ folder
